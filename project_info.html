DASNA Page Management System (PMS)

What is it?
The DASNA page management system is a simple set of webpages designed to aid a single person in managing and maintaining the main DASNA homepage, located at http://www.dasna.net.

How can I use it?
  1. First, create a new account to use with the page editing system, by visiting http://www.dasna.net/ssl/beta/reset.php.
    Here, you must:
    • Enter your provided username, and press enter. Note: New users cannot be created!
    • Enter email or phone number to send confirmation code to.
    • Select your preffered method to receive confirmation code (email, or mobile carrier) from the dropdown list-box. Confirmation-codes via e-mail are slow. SMS (text message) is preffered, but device must have mobile data connection enabled.
    • Enter the confirmation code received, and press enter.
    • Enter new desired password (twice), and press enter.
    • Done.
  2. After successfully creating a user in the above step, navigate to http://login.dasna.net. Note: You MUST use Chrome!!
    Here, you can:
    • Note: This page will automatically redirect to http://www.dasna.net/ssl/beta/reset.php (or wherever the current version is).
    • Enter username, then press submit.
    • Enter password, then press submit.
    • If successfully verified, a page should appear with a word-processor-like interface (if login fails, please try again).
  3. You are now successfully logged into the DASNA Page Editing System.
    Please, keep in mind:
    • Any keystrokes typed into the editing window will reflect live on the page (I designed it to save every time a key is released). Note: Currently, there is no save/restore feature, but implementation has already started
    • The dropdown listbox in the top left-hand corner selects the column on the page to edit. Selecting an item from the dropdown listbox automatically changes the page being edited in the editor.
    • To see your changes, view them at: http://www.dasna.net/beta.php
